## Installation

All examples require `@humanspeak/svelte-motion`.

```bash
npm install @humanspeak/svelte-motion
```

Import what you need:

```svelte
<script lang="ts">
    import { motion, AnimatePresence, type Variants } from '@humanspeak/svelte-motion'
</script>
```

<Note>
  Every example on these pages uses real code from the Svelte Motion source. Nothing is invented — patterns map directly to the library's tested behavior.
</Note>
